### Hello-world bot for telegram

#### How to deploy

This bot will be deployed into https://dashboard.heroku.com/apps/artem-hello-bot

Deploy:
`git push heroku master`

Logs:
`heroku logs --tail`

More commands: https://devcenter.heroku.com/articles/heroku-cli-commands
